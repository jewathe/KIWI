POST/ingredients?token=userToken

Creer un ingredient

Parametres de la requette

Parametres Types Requis Description
token String OUI Token d'identification de l'utilisateur

Parametres du BODY de la requette

              
             $ {
                    "adoptedTitle": "",
                    "proposedTitle": "Tomate",
                    "categories": [idCategorie1,idCategorie2,idCategorie3],
                    "authorId": 3,
                }
Attributs Valeurs Defaut
adoptedTitle Titre qui sera adoptee pour l'ingredient une fois qu'elle sera validee ""
proposedTitle Titre propose par le createur de l'ingredient
categories Liste des categories de l'ingredient []

Exemple de reponse

         
             $ {
                    "id": 2,
                    "adoptedTitle": "",
                    "proposedTitle": "Tomate",
                    "idCategories": [1,3,2],
                    "authorId": 4,
                }
