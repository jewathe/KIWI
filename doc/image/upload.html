POST/recipes/:id/upload/?token=userToken

Ajouter d' images a une recette

Parametres de la requette

Parametres Types Requis Description
id int OUI identifiant de la recette sur laquelle sera effectuee l'ajout de d'images
token String OUI Token d'identification de l'utilisateur createur de la recette
